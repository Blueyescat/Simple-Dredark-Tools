  • Chat Highlighter


    (preview) Every line is a different text to match and highlight. Press Enter for a new line. It only works for user and system messages.
    Advanced - RegEx It is possible to use regex by surrounding a line with /. Use it if you know how.
    Some useful examples:
    • /foo(?! ?bar)/ - highlight "foo" that doesn't end with "bar" or " bar"
    • /(?<!baz)foo(?! ?bar)/ - highlight "foo" that doesn't start with "baz" and doesn't end with "bar" or " bar"
    • /(?<!baz|quu?x ?|bar)foo/ - highlight "foo" that doesn't start with "baz", "qux", "qux ", "quux", "quux " or "bar"

The options are automatically saved.
